# Vite template

   一个集成Vite、vue3、typescript、axios、sass的简单项目模板。

## 安装

```shell
npm install
```

## 运行

```shell
npm run dev
```

## 编译

```shell
npm run build
```

## 说明
* server: 可在vite.config.js中配置           
   <kbd>host</kbd> : 127.0.0.1               
   <kbd>post</kbd> : 9527           
* .env相关：在config文件夹中配置
